# Micro-Sniper 管理页面使用说明

## 访问地址

启动服务后，访问：`http://localhost:8000/static/`

## 功能说明

### 1. 登录

- 输入您的 API Key（从 `/identity/api-keys` 创建）
- API Key 会保存在浏览器的 localStorage 中
- 下次访问自动登录

### 2. 任务管理

#### 创建创作者监控任务
- 输入创作者 ID（每行一个）
- 设置监控天数（1-30天）
- 点击创建

#### 创建趋势分析任务
- 输入关键词（每行一个）
- 系统会分析关键词的爆款趋势
- 点击创建

#### 查看任务列表
- 查看所有任务的状态
- 实时刷新任务进度

### 3. 连接器操作

#### 内容提取
- 选择平台（小红书/微信/通用网站）
- 输入 URL 列表（每行一个）
- 提取 AI 分析的内容摘要

#### 采收内容
- 选择平台
- 输入创作者 ID 列表
- 设置数量限制
- 批量采收用户主页内容

#### 搜索提取
- 选择平台
- 输入关键词列表
- 设置结果数量限制
- 搜索并提取内容

### 4. 平台列表

- 查看所有支持的平台
- 查看每个平台的功能特性
- 查看频率限制配置

## 技术栈

- **前端框架**: 纯 HTML + Bootstrap 5
- **HTTP 客户端**: Axios
- **样式**: 简洁的海外风格（Vercel/Stripe 风格）
- **图标**: Bootstrap Icons

## 文件结构

```
static/
├── index.html          # 主页面
├── js/
│   └── app.js         # JavaScript 逻辑
└── css/               # （可扩展自定义样式）
```

## 特点

✨ **简洁设计**: 海外风格，清爽简洁
✨ **实时更新**: 任务进度实时显示
✨ **卡片展示**: 结果以卡片形式展示
✨ **响应式**: 支持移动端和桌面端
✨ **无需编译**: 纯静态文件，开箱即用
